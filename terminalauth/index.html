Emacs Quick Reference

Open file: emacs file.txt

Exit Emacs: C-x C-c

Save files: C-x s

Write current file: C-x C-w

Split Screen Vertically: C-x 2

Split Screen Horizontally: C-x 3

Switch buffer: C-x o

Git Quick Reference

Git Workflow Introduction

Add files to be committed: git add filename

Commit changes with message: git commit -m "message"

Push changes: git push

See current changes: git status

Pull changes: git pull
